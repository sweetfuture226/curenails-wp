Page Animations And Transition is provide multiple Animation effect to your WordPress site. Show your page with stylish transition. 

== Description ==

Page Animations And Transition that is available for free to use. This Plugin has multiple Animation and css3 transition Effect . Using this plugin your WordPress page will be load with effective animation styles

Page Animations And Transition features:

* Set Page In Animation Effect
* Set Page Out Animation Effect
* Set Page In Animation Duration (Speed) 
* Set Page Out Animation Duration (Speed) 

### 9 different Animation effects available:

* Fade
* Fade Down
* Fade Up
* Fade Left
* Fade Right
* Rotate
* Flip X
* Flip Y
* Zoom
* Overlay Slider Left and Right
* Site Preloader Settings
* Exclude Page or Post

Plugin by <a href="https://weblizar.com">Weblizar</a>

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `page-animations-and-transitions` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Ready, now you can enjoy it, and if you like it and find it useful.
